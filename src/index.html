Loot Bidding

Configure guild loot message patterns. Settings are global (not per-character).

BIDS OPEN KEYWORD
Keyword in guild message that opens bidding on a list of items.
MEMBER STATUSES (priority order, comma-separated)
First listed = highest priority. Members beat applicants/alts even with lower bids.
CLOSING KEYWORD
Start of countdown message, e.g. "CLOSING IN 15s!" or "CLOSING IN LAST CALL!"
SOLD KEYWORD
Clears all active bids from the panel when seen.
BIDS OPEN ITEM SEPARATOR
Separates item names in the bids-open announcement (e.g. comma).
STATUS MESSAGE ITEM DELIMITER
Separates items in CLOSING IN and SOLD messages (e.g. " // ").
Equipment & Desired Loot

Per-character. Load your CharacterName-Inventory.txt to auto-populate current gear. Add desired items to receive voice alerts when they're put up for bid.

Alpha Loot Tracker

Tracks which group member is next for alpha loot. Rotation advances automatically when a listed item is looted. The rotation bar appears in the Group panel while you're in a group.

Example Message Formats
Bids open:  Stardrakin tells the guild, 'Item One, Item Two - BIDS OPEN'
Player bid:  Nightwish tells the guild, 'Soul Essence of Aten Ha Ra mem 125'
Countdown:  Balthemal tells the guild, 'CLOSING IN 15s! Item (Winner - 125 dkp) // ...'
Sold:  Balthemal tells the guild, 'SOLD! Item (Winner - 125 dkp) // ...'
